# FTF Modpack Collaboration

A TACZ-based zombie apocalypse / firearms survival pack built around **Timeless and Classics: Zero**, with a curated set of tactical gear, vehicles, structures, and horde-survival content layered on top.

Built for **Minecraft 1.20.1 / Forge 47.4.10**.

## Highlights

- **Guns & combat:** TaCZ core + LesRaisins Tactical Equipment, LesRaisins Armor, Point Blank: Jelly, Fracture Point, Just Gunpowder, Enhanced Arsenal, Simple Missiles
- **Zombies & hordes:** The Hordes, Infectious, Enhanced Hordes, Undead Revamped, Outbreak, plus Zombies More content
- **Vehicles:** Immersive Aircraft, Civilian Aviation
- **Exploration & building:** Create, Additional Structures, Moog's Voyager Structures, Valhelsia Structures, The Lost Cities, Waystones, JourneyMap
- **Storage & tech:** Applied Energistics 2, Mekanism (+ Generators), Sophisticated Backpacks/Core, Iron Chests, Tom's Storage, Simple Storage Network
- **Quality of life:** JEI, AppleSkin, Leawind's Third Person, ImmediatelyFast, Embeddium + Oculus for performance
- **Visuals:** Fresh Animations, Serene Seasons (+ Plus), Immersive Weathering, Dynamic Waters

Also included: shaderpacks (Bliss, Complementary Reimagined/Unbound, Insanity, Photon, Solas) and resourcepacks (Fresh Animations, TZP, Panorama with Shaders).

## Installing

Grab the latest `.mrpack` from the [Releases page](https://github.com/ViperX1919/FTF-Modpack-Collaboration/releases/latest) and import it with the [Modrinth App](https://modrinth.com/app) (Import from file) or drag it onto [Prism Launcher](https://prismlauncher.org/).

## Notes

- Managed with [packwiz](https://packwiz.infra.link/) — mods are tracked as lightweight metadata rather than committed jars, so future updates stay easy to diff and collaborate on.
- A handful of mods (custom TACZ addons, a few CurseForge "no third-party distribution" mods) are bundled directly in the pack since they aren't auto-downloadable — these won't show individual update notifications but are included and working.
